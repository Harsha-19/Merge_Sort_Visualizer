# Merge Sort Visualizer & Educational Platform

A modern, SaaS-quality educational web application that teaches the Merge Sort algorithm using step-by-step visual animation, real-world analogies, and interactive controls.

## Tech Stack
- Frontend: React + Vite, Tailwind CSS, Framer Motion, Zustand
- Backend: Node.js + Express (Provides quiz data)

## Setup Instructions

### 1. Start the Backend
Open a terminal and run the following commands:
```bash
cd backend
npm install
npm start
```
The backend server will start on port 5000.

### 2. Start the Frontend
Open a new terminal and run the following commands:
```bash
cd frontend
npm install
npm run dev
```
The Vite development server will start on port 5173.

## Usage
- Enter a custom array (comma-separated integers up to 16 elements).
- Use the play, pause, step forward, and step backward buttons to navigate the animation.
- Adjust animation speed using the slider.
- Read the real-world analogy and algorithm explanations updated in real-time.
- Take the interactive quiz at the bottom to test your knowledge!
